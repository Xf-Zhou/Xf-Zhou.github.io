1、什么是JavaScript

1.1、概述

JavaScript（简称“JS”） 是一种具有函数优先的轻量级，解释型或即时编译型的高级编程语言。JS是一门世界上最流行的脚本语言。

1.2历史

布兰登·艾奇对Java一点兴趣也没有。为了应付公司安排的任务，他只用10天时间就把Javascript设计出来了。

ECMAScript可以理解为是JavaScript的一个标准

最新版本已经到了es6版本,但是大部分浏览器还只停留在支持es5代码上！

开发环境—线上环境，版本不一致。

2.3、数据类型

数值、文本、图形、音频、视频……

  1. number

    js不区分小数和整数，都是number

123 //整数123
123.1  //浮点数123.1
1.123e3 //科学计数法
-99
NaN  //not a number
Infinity  //无限大

  1. 字符串

    ‘abc’ “abc”

  2. 变量

    var nam = "adfas"

  3. 布尔值

    true false

  4. 逻辑运算

    &&
||
!

  5. 比较运算符

    =  赋值
== 等于（类型不一样，值一样，也会判断为true
===  绝对等于（类型一样，值一样，结果才为true

    注意：

    ​ NaN===NaN，这个与所有的数值都不相等，包括自己

    ​ 只能通过isNaN(NaN)来判断这个数是否是NaN

  6. 浮点数问题

    console.log((1/3)===(1-2/3)); //false
console.log(Math.abs(1/3-(1-2/3))<0.0000001); //true

    尽量避免使用浮点数进行运算，存在精度问题！

  7. null 和underdefined

    null 空 underfined 未定义的

  8. 数组

    可存储不同类型的元素（类似于python的list），下表越界会报未定义。

    var arr = [1,2,"hello",null,true];

  9. 对象

    对象是大括号，数组是中括号。

    每个属性使用逗号隔开，最后一个不需要添加。

    var person = {
    name:"suda",
    age:3,
    tags:['js','java','web','...']
}

    取对象的值

    person.name
"suda"
person.age
